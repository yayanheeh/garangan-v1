Introduction

Garangan CSS adalah kerangka kerja CSS yang ringan, responsif, dan modern untuk pengembangan yang lebih cepat dan dapat diperluas.

Garangan CSS memberikan gaya dasar untuk tipografi dan elemen, sistem tata letak responsif berbasis flexbox, komponen dan utilitas CSS murni dengan koding praktik terbaik dan bahasa desain yang konsisten.
